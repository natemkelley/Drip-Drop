About the Project

This project was inspired by the trend of smart home appliances that are popping up everywhere! In my opinion, a smart home is one that eliminates waste. That is why I built DripDrop!

Link to other projects
NodeJS

The charts, analytics, and calls to external APIs are all being handled using this technology.

Firebase

To showcase how this project can be integrated into IoT, the database is called via API. The Firebase database is updated every 5 seconds with incoming data from the flow meter and then called back down to run analytics.

Python

Python is used to control both the solenoid which stops and starts the water. Python is also using to gather data from the flow meter.
